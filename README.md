# hardhat-tracer 🕵️

Allows you to see emitted events

## Installation

**Step 1:** Install the package

```
npm i hardhat-tracer
```

**Step 2:** Add to your `hardhat.config.js` file

```
require("hardhat-tracer");
```

## Usage

```
yarn test --logs
```
